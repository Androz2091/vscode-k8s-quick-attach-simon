# vscode-k8s-quick-attach

This extension provide a quick way to attach VSCode to k8s Pod

## Demo

![](https://raw.githubusercontent.com/mtsmfm/vscode-k8s-quick-attach/main/demo.gif)

## Commands

```
k8s quick attach: Quick Attach k8s Pod
```
